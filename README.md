Weather Dashboard Pro

Project Overview
A full-stack weather forecasting application developed by a team of 6 members. The project integrates real-time weather data with a custom Machine Learning model to predict atmospheric conditions. It features a modern, glassmorphism-styled web interface connected to a Python Flask backend.

Repository Structure

app.py: The main Flask application handling backend logic and API routes.

pkl files: The trained Random Forest model, encoder, and scaler used for predictions.

weather_forecast_data.csv: The dataset used to train the machine learning model.

templates and static folders: The HTML, CSS, and JavaScript files for the user interface.

Main Features

Machine learning predictions using Scikit-Learn.

Live weather data integration via the Open-Meteo API.

Responsive frontend with interactive charts and dynamic weather gauges.

Tech Stack

Python, Flask, Scikit-Learn, Pandas

HTML, CSS, JavaScript
